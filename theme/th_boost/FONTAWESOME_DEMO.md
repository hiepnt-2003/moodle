## Cách thêm vào Moodle:

1. Đăng nhập với tài khoản Admin
2. Bật chế độ "Turn editing on"
3. Click "Add a block" → chọn "HTML"
4. Paste code HTML ở trên vào block
5. Save changes

Theme TH Boost sẽ tự động style các icons này!
